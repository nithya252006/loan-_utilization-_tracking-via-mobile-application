Frontend Environment Setup

Project Title

**Loan Tracking and Utilization Monitoring Application**

Objective

The frontend of the Loan Tracking and Utilization Monitoring System is developed using HTML, CSS, and JavaScript to provide a responsive and user-friendly interface for managing loan information, tracking repayments, and monitoring loan utilization.

Technologies Used

* HTML
* CSS
* JavaScript 
* Visual Studio Code (VS Code)
* Git & GitHub
* Google Chrome Browser

 Software Requirements

| Software           | Purpose             |
| ------------------ | ------------------- |
| Visual Studio Code | Code Development    |
| Google Chrome      | Application Testing |
| Git                | Version Control     |
| GitHub             | Repository Hosting  |

Setup Procedure

 Step 1: Install Visual Studio Code

Download and install Visual Studio Code.

Step 2: Create Project Folder

Create a project folder named:

loan-tracking-system

Step 3: Create Frontend Files

Create the following files:

* index.html
* dashboard.html
* loan-form.html
* loans.html
* payments.html
* reports.html
* css/style.css
* js/script.js

Step 4: Open Project in VS Code

Open the project folder using Visual Studio Code.

Step 5: Run the Application

Open the HTML files in a web browser or use the VS Code Live Server extension for local development and testing.

 Frontend Features

* User Login Interface
* Dashboard Overview
* Loan Application Form
* Loan Tracking Module
* Repayment Monitoring
* Reports and Analytics
* Responsive Navigation System

Expected Outcome

The frontend provides an intuitive interface for users to manage loans, track repayments, monitor loan utilization, and view reports efficiently through a modern web-based system.
