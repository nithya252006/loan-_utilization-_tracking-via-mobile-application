 Dashboard UI Design

 Project

Loan Tracking and Utilization Monitoring Application

 Purpose

The Dashboard provides users with a quick overview of loan utilization, expenses, reports, and reminders.
 Beneficiary Dashboard

 UI Components

* Welcome Message
* Loan Summary Card
* Expense Summary Card
* Remaining Balance Card
* Quick Access Buttons

 Wireframe
```text
+----------------------------------+
| Dashboard                        |
+----------------------------------+
| Welcome, User                    |
+----------------------------------+
| Loan Amount      : ₹50,000       |
| Total Expense    : ₹15,000       |
| Remaining Balance: ₹35,000       |
+----------------------------------+
| [ Loan Details ]                 |
| [ Add Expense ]                  |
| [ Upload Proof ]                 |
| [ Reports ]                      |
| [ Reminders ]                    |
+----------------------------------+

```
Features

* View Loan Details
* Add Expenses
* Upload Proof Documents
* View Reports
* Check Reminders
